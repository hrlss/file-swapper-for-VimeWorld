# file swapper for VimeWorld
A lightweight file swapper for VimeWorld

## Features
- Switches between Default and Modified client files
- Supports .jar and .dll file replacement
- Auto-detects AppData VimeWorld directory

## Usage
Run `python swapper.py` and select an option:
1 — Load Default
2 — Load Modified
0 — Exit

## Folder Structure
- Default/   ← original files
- Modified/  ← modded files
